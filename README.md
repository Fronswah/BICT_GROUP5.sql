## Project Overview

This project implements a Public Health Clinic Records System using MySQL. The system was developed as part of COMP 102 at Limkokwing University of Creative Technology, Sierra Leone.

The database supports the management of:

* Clinics
* Staff
* Patients
* Appointments
* Medical Records
* Medicines
* Prescriptions
* Billing

The project contributes to SDG 3: Good Health and Well-being by improving healthcare record management through a structured digital database system.
